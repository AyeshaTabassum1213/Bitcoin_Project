# Bitcoin_Project
An interactive Bitcoin Price Tracker built with HTML, CSS, and JavaScript. This project fetches real-time Bitcoin price data using the CoinDesk API and displays it in a clean, responsive interface.
Features

Real-time Bitcoin price tracking

Fetches live data from the CoinDesk API

Built with Next.js and React

Clean and minimal UI

Easy to set up and run locally

Tech Stack

Next.js

React

JavaScript (ES6+)

CoinDesk API

Usage

The homepage will display the current Bitcoin price in different currencies (USD, GBP, EUR).

Data is updated automatically by fetching from the CoinDesk API.
